# PowerBI_report_-_kickstart_crowdfunding

Introduction of the project: Crowdfunding enables projects to be funded by small contributions from many people online. Kickstarter is a popular platform where creators independently raise funds and offer rewards to backers.

Purpose: The reports provide insights into around 364K projects created between January 2009 and December 2019 across 22 countries and 169 categories, with successful projects analysed separately.
